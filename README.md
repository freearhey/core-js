# @freearhey/core

## Installation

```sh
npm install @freearhey/core
```

## Usage

```js
import { Collection } from '@freearhey/core'

const bears = new Collection(['polar', 'panda', 'brown'])

bears.sort()

console.log(brears.all()) // ['brown', 'panda', 'polar']
```
